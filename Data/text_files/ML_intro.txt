
    Machine Learning Basics

Machine Learning is a subset of Artificial Intelligence that enables systems to learn from data without being explicitly programmed.

Types of Machine Learning:
- Supervised Learning
- Unsupervised Learning
- Reinforcement Learning

Common Algorithms:
- Linear Regression
- Decision Trees
- Random Forest
- Support Vector Machines

Machine learning is used in recommendation systems, fraud detection, image recognition, and predictive analytics.
